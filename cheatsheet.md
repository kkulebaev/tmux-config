# tmux cheatsheet

Префикс — **Ctrl+a**.

| Keys | Действие |
| --- | --- |
| `Ctrl+a %` | Сплит вертикально (новая панель справа) |
| `Ctrl+a "` | Сплит горизонтально (новая панель снизу) |
| `Ctrl+a ←/↑/↓/→` | Перейти в соседнюю панель стрелками |
| `Ctrl+a &` | Убить окно (с подтверждением `y/n`) |
