# tmux cheatsheet

Префикс — **Ctrl+a**.

**Окно** (window) — вкладка в статус-баре снизу. **Панель** (pane) — отдельный shell внутри окна; сплиты создают новые панели в том же окне.

| Keys | Действие |
| --- | --- |
| `Ctrl+a %` | Сплит вертикально (новая панель справа) |
| `Ctrl+a "` | Сплит горизонтально (новая панель снизу) |
| `Ctrl+a ←/↑/↓/→` | Перейти в соседнюю панель стрелками |
| `Ctrl+a x` | Убить текущую панель (без подтверждения) |
| `Ctrl+a T` | Переименовать текущую панель (заголовок появится в полоске сверху) |
| `Ctrl+a ,` | Переименовать окно (Enter — подтвердить, Esc — отменить) |
| `Ctrl+a &` | Убить окно целиком со всеми панелями (с подтверждением `y/n`) |
